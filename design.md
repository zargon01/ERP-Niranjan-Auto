# 🎨 Design System — ERP Niranjan Enterprises

## 📌 Overview

This document defines the **UI/UX standards, design system, and development norms** for the ERP system used by **Niranjan Enterprises**, an automobile parts manufacturing company.

This is an **industrial ERP dashboard**, not a marketing product.

---

## 🎯 Design Philosophy

- Clarity > Visual aesthetics  
- Speed > Animation  
- Data density > Whitespace  
- Consistency > Creativity  

### Core Rules

- Minimize cognitive load  
- Max 3-click navigation  
- Always prioritize readability  
- Every UI element must serve a purpose

### Color Usage Rules

#### ✅ DO:
- Use orange for actions (buttons, highlights)
- Use dark colors for structure (sidebar/header)
- Use white/light backgrounds for readability

#### ❌ DON'T:
- Use orange as background
- Use gradients
- Mix multiple dark shades randomly
